# Art Direction & Theme Guidance

- Use a limited palette (3-5 primary colours + neutrals). Define tokens in a theme config file.
- Typography: single display font + single UI font; provide sizes and scale steps.
- Iconography: cohesive icon set (SVG or vector) with consistent stroke widths.
- Recommend using curated UI asset packs as a starting point rather than designing from scratch.
- All visual decisions should be captured as tokens, not hardcoded — this makes theme changes a config swap.
