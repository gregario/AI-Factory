# Art Direction & Theme Guidance
- Recommend using curated UI packs (e.g., Kenney UI packs) as base assets.
- Use a limited palette (3–5 primary colors + neutrals). Define tokens in `templates/ui_theme.json`.
- Typography: single display font + single UI font; provide sizes and scale steps.
- Iconography: cohesive icon set (SVG or 9-slice) and consistent stroke widths.
- Asset pipeline: store source art in `/projects/<project>/assets/source/` and exported runtime in `/projects/<project>/assets/ui/`.
